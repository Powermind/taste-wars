# Taste Wars ⭐

**Taste Wars** is a fan-made web application that lets you rate Star Wars movies and series, generate a “Taste Code,” and compare your taste with friends.  
> ⚠️ This app is **not affiliated with or endorsed by Lucasfilm or Disney**. Star Wars® is a trademark of Lucasfilm Ltd.


---

## Features

- Rate all Star Wars movies and series (0–10)  
- Randomize or reset your ratings  
- Generate a personal **Taste Code**  
- Share your Taste Code via link  
- Load your own or a friend’s Taste Code  
- Compare ratings with a friend using:
  - Match percentage  
  - Confidence percentage  
  - Radar chart visualization  
- Fully responsive and mobile-friendly
